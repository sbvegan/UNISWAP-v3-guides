# Uniswap V3 Guides

I'm running through these guides to get my feet wet with Uniswap.

## Initializing a project

```
npm init
```

The dependencies

```
npm i typescript --save
npm i ts-node --save
npm i @uniswap/v3-sdk --save
npm i @uniswap/sdk-core i --save
npm i ethers  --save
npm install -D tslib @types/node
```
